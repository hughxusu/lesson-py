## 课程目标

学习Python编程语言，以及相关应用。

## 课程内容

* 了解Python编程语言的由来与应用。
* 配置Python的开发环境。
* 掌握Python的基本语法。
* 了解计算机程序的基本运行原理。
* 学习面向对象编程的思想。
* 学习Python的常用库，如：文件操作、Numpy等。
