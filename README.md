## 课程目标

学习Python编程语言。

## 课程内容

* 了解Python在互联网行业中的应用。
* 了解计算机程序的基本运行原理。
* 安装Python开发环境与开发工具。
* 学习Python基本语法和高级特性。
* 学习面向对象编程的思想。
* 学习Python的标准库库，如：文件操作等。
