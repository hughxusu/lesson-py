## 课程目标

学习 python 编程语言。

## 课程内容

* 了解 python 在互联网行业中的应用。
* 了解计算机程序的基本运行原理。
* 安装 python 开发环境与开发工具。
* 学习 python 基本语法和高级特性。
* 学习面向对象编程的思想。
* 学习 python 的标准库库，如：文件操作等。
