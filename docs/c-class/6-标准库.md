# 常用的标准库类

Python标准库（Standard Library）是随Python解释器一起安装的“内置工具箱”，能完成文件处理、网络通信、数学计算、甚至是简单的图形界面开发。

## `datetime`

`datetime`是Python中用于处理时间的标准库，`datetime`中最常用的类是

1. `date`处理年、月、日。

```python
from datetime import date

now = date.today()
print(f"现在是：{now}")
print(f"现在是：{now.year}年{now.month}月{now.day}日")
```

2. `datetime`用于处理年、月、日、时、分、秒；`time`处理时间，一般不单独使用，而与`datetime`结合使用。

```python
from datetime import datetime

now = datetime.now()
print(f"{type(now)}")
print(f"日期是：{now.year}年{now.month}月{now.day}日")
print(f"时间是：{now.hour}时{now.minute}分{now.second}秒")

date = now.date()
print(f"date类型：{type(date)}")

time = now.time()
print(f"time类型：{type(time)}")

time_str = now.strftime('%Y-%m-%d %H:%M:%S')
print(f"{time_str} 类型：{type(time_str)}")

date_str = "2026-03-28 14:30:00"
date = datetime.strptime(date_str, "%Y-%m-%d %H:%M:%S")
print(f"{date}")
```

* `strftime`可以实现字符串与时间的相互转换。

3. `timedelta`用于计算时间间隔

```python
from datetime import timedelta, datetime

now = datetime.now()
print(f"现在是：{now}")

future = now + timedelta(days=10)
print(f"10天后是：{future}")

past = now - timedelta(hours=5)
print(f"5小时前是：{past}")
```

## `Enum`

通过`Enum`类可以定义枚举类型，枚举成员是单例。枚举的主要作用：提高代码可读性**、**增强安全性、易于维护和防止逻辑错误。

```python
from enum import Enum, auto

class Status(Enum):
    PENDING = auto()
    RUNNING = auto()
    SUCCESS = auto()
    FAILED = auto()

status = Status.PENDING
print(status)
print(f'状态值是: {status.value}, 状态名称是: {status.name}')

status = Status.RUNNING
print(f'状态值是: {status.value}, 状态名称是: {status.name}')
```

枚举值可以自定义，自定义的枚举值可以使任意类型。

```python
from enum import Enum

class Color(Enum):
    RED = 1
    GREEN = 'green'
    BLUE = 'blue'

color = Color.RED
print(color)
```

## `collections`

Python内置的`collections`模块提供了一系列高性能的特殊容器数据类型。

1. 双端队列`deque`，可以在队列的头和尾，插入和弹出元素。

```python
from collections import deque

deq = deque()

# 添加至队尾
deq.append(2)    
deq.append(4)

# 添加至队首
deq.appendleft(3)  
deq.appendleft(1)

for item in deq:
    print(item, end=' ')
print()

# 弹出队尾元素
print(f"弹出队尾元素: {deq.pop()}") 

# 弹出队首元素 
print(f"弹出队首元素: {deq.popleft()}")  
```

2. `Counter`用于统计可迭代对象中元素的出现次数。

```python
from collections import Counter

colors_one = ['red', 'green', 'blue', 'green']
cnt_one = Counter(colors_one)
print(cnt_one)
print(f"green 出现了 {cnt_one['green']}")

colors_two = ['red', 'yellow', 'green']
cnt_two = Counter(colors_two)
print(cnt_two)

print(f"并集: {cnt_one + cnt_two}")
print(f"差集: {cnt_one - cnt_two}")
print(f"交集: {cnt_one & cnt_two}")
print(f"并集: {cnt_one | cnt_two}")
```

* `Counter`可以像`set`集合一样，进行集合运算。

## 正则表达式

正则表达式是用一套特定的字符组合成一个“规则字符串”，用来在海量文字中快速找到、替换或验证符合特定特征的内容。Python中的`re`模块可以处理正则表达式。

使用[`re.match`](https://docs.python.org/zh-cn/3.13/library/re.html#re.match)从字符串的头部开始查找，找到一个匹配的结果就返回。

* 如果没有匹配结果，返回`None`。
* 如果匹配到结果，返回[`re.Match`](https://docs.python.org/zh-cn/3.13/library/re.html#re.Match)对象，`Match`对象的常用方法：
  * `group()`提取匹配到的数据。
  * `start()`获取子串在主串中的起始位置。
  * `end()`获取子串在主串中的结束位置。
  * `span()`返回下标序号`(start, end)`，如果没有匹配成功返回 `(-1, -1)` 。

```python
import re

pattern = r'hello'
s = 'hello hello'
result = re.match(pattern, s)
print(result)
print(type(result))

print(result.group())
print(result.start())
print(result.end())
print(result.span())
```

* `pattern = r'hello'`中的`r`表示原始字符串，不要处理`\x`的转义字符。

> [!note]
>
> 处理正则时，正则表达式采用`r'xxx'`原始字符串。

### 正则表达规则

1. 表示字符

| 字符 | 功能                                               |
| ---- | -------------------------------------------------- |
| `.`  | 匹配任意1个字符（除了\n）                          |
| `[]` | 有限集合，匹配`[]`中列举的字符                     |
| `\d` | 匹配数字（0-9）                                    |
| `\D` | 匹配非数字                                         |
| `\s` | 匹配空白（空格、tab、`\n`、`\r`）                  |
| `\S` | 匹配非空白                                         |
| `\w` | 匹配单词字符（a-z、A-Z、0-9、_），满足变量名规则。 |
| `\W` | 匹配非单词字符                                     |

```python
# 表示任意字符
print(re.match(r'.', 'a').group())
print(re.match(r'...', 'xyz').group())

# 表示任意数字
print(re.match(r'\d', '1').group())

# 表示任意非数字字符
print(re.match(r'\D', 'a').group())

# 表示a-z和5-9范围内取
print(re.match('[a-z5-9]', 'a8').group())
```

2. 表示数量

| 字符    | 功能                                               |
| ------- | -------------------------------------------------- |
| `*`     | 匹配一个字符出现0次或无限多次，即一个字符可有可无  |
| `+`     | 匹配一个字符出现1次或无限多次，即一个字符至少有1次 |
| `?`     | 匹配一个字符出现1次或0次，即至多一次               |
| `{m}`   | 匹配一个字符出现m次                                |
| `{m,}`  | 匹配一个字符至少出现m次                            |
| `{m,n}` | 匹配一个字符出现m到n次                             |

```python
# 匹配0个或多个字符串
print(re.match(r'\D*', 'abc').group())

# 匹配至少1个数字
print(re.match(r'\d+', '123abc').group())

# 匹配电话号码
print(re.match('1[34578]\d{9}', '13144396831').group())
```

3. 表示边界

| 字符 | 功能                                               |
| ---- | -------------------------------------------------- |
| `^`  | 匹配字符串开头                                     |
| `$`  | 匹配字符串结尾                                     |
| `\b` | 匹配一个单词的边界，仅表示边界信息，不包含字符本身 |
| `\B` | 匹配非单词边界                                     |

```python
# 匹配完整的电话号码
print(re.match(r'1[34578]\d{9}$', '13144396831').group())
print(re.match(r'\w+o\b', 'hello').group())
```

4. 匹配分组

| 字符         | 功能                                 |
| ------------ | ------------------------------------ |
| `|`          | 匹配左右任意一个表达式               |
| `(ab)`       | 将括号中的字符作为一个分组           |
| `\num`       | 取得分组索引，\2表示第二个括号的内容 |
| `(?P<name>)` | 分组起别名                           |
| `(?P=name)`  | 使用别名                             |

```python
# 匹配0-100之间的数字
print(re.match(r'[1-9]\d?$|0$|100$', '88').group())

# 匹配邮箱
print(
    re.match(
        r'\w+@(163|126|gmail|qq).(com|cn|net)$', 
        'tommy@qq.com'
    ).group()
)
```

5. 正则表达式中文字符范围
   * 匹配中文字符的正则表达式：`[\u4e00-\u9fa5]`。
   * 匹配双字节字符（包括汉字在内）：`[^\x00-\xff]`。

```python
# 中文匹配
print(
    re.match(
        r'[\u4e00-\u9fa5]{5}', '青山横北郭，白水绕东城。'
    ).group()
)
```

6. 贪婪模式

   * 贪婪模式：总是尝试匹配尽可能多的字符。

   * 非贪婪模式：总是尝试匹配尽可能少的字符。

   * 默认状态是贪婪的，如果需要关闭贪婪模式，使用`?`。

```python
# 贪婪模式
print(
    re.match(
        r'<div>.*</div>', '<div>Python</div><div>Regex</div>'
    ).group()
)

# 非贪婪模式
print(
    re.match(
        r'<div>.*?</div>', '<div>Python</div><div>Regex</div>'
    ).group()
)
```

### 正则表达式常用方法

1. [`re.search()`](https://docs.python.org/zh-cn/3.13/library/re.html#re.search)查找字符串的任意位置，返回第一个匹配结果，返回`re.Match`对象。

```python
result = re.search(r'白水', '青山横北郭，白水绕东城。')
print(type(result))
print(result.group())

result = re.search(r'^白水', '青山横北郭，白水绕东城。')
print(result)
```

2. [`re.findall()`](https://docs.python.org/zh-cn/3.13/library/re.html#re.findall)以列表形式返回全部能匹配的子串，如果没有匹配，则返回一个空列表。

```python
result = re.findall(r'\d+', 'hello 12345 6789')
print(result)
```

3. [`re.finditer()`](https://docs.python.org/zh-cn/3.13/library/re.html#re.finditer)与`findall`的类似，但它返回一个包含所有匹配结果（`re.Match`对象）的迭代器。

```python
result = re.finditer(r'\d+', 'hello 12345 6789')
print(result)
for item in result:
    print(item)
```

4. [`re.split()`](https://docs.python.org/zh-cn/3.13/library/re.html#re.split)按照能够匹配的子串，将字符串分割后返回列表。

```python
result = re.split(r'[\s]+', 'a b c d')
print(result)
```

5. [`re.sub()`](https://docs.python.org/zh-cn/3.13/library/re.html#re.sub)可以替换掉匹配的模式串，`re.sub(pattern, repl, string, count=0)`
   * `pattern`模式串。
   * `repl`替换后的内容，可以是字符串，也可以是一个函数。
   * `string`待处理的原始字符串。
   * `count`替换次数。

```python
result = re.sub(r'\s+', '-', 'a  b c d', count=2)
print(result)
```

### 正则对象

`re.compile()`可以将正则表达式的样式编译为一个正则对象[`re.Pattern()`](https://docs.python.org/zh-cn/3.13/library/re.html#re.Pattern)。这个正则对象包括的方法，与前面的相同：

* `Pattern.search()`
* `Pattern.match()`
* `Pattern.findall()`
* `Pattern.finditer()`
* `Pattern.sub()`

```python
import re

result = re.compile(r'\d+')
print(result)
print(type(result))

src = 'a 123 b 456 c 789 d'

print(result.match(src))
print(result.search(src))
print(result.findall(src))
print(result.finditer(src))
print(result.split(src))
print(result.sub('-', src))
```
