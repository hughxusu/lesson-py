# 标准库的常用类

Python标准库（Standard Library）是随Python解释器一起安装的“内置工具箱”，能完成文件处理、网络通信、数学计算、甚至是简单的图形界面开发。

## `datetime`

`datetime`是Python中用于处理时间的标准库，`datetime`可以获得从1970年1月1日00:00:00开始到现在为止的毫秒数。

## 正则表达式

[正则表达式](https://www.runoob.com/regexp/regexp-tutorial.html)是用一套特定的字符组合成一个“规则字符串”，用来在海量文字中快速找到、替换或验证符合特定特征的内容，比如:邮箱、图片地址、手机号码等。正则表达式的特点：

* 语法很令人头疼，可读性差。
* 通用行很强，能够适用于很多编程语言。

Python中的`re`模块可以处理正则表达式，使用[`re.match`](https://docs.python.org/zh-cn/3.13/library/re.html#re.match)从字符串的头部开始查找，找到一个匹配的结果就返回。

* 如果没有匹配结果，返回`None`。
* 如果匹配到结果，返回[`re.Match`](https://docs.python.org/zh-cn/3.13/library/re.html#re.Match)对象，`Match`对象的常用方法：
  * `group()`提取匹配到的数据。
  * `start()`获取子串在主串中的起始位置。
  * `end()`获取子串在主串中的结束位置。
  * `span()`返回下标序号`(start, end)`，如果没有匹配成功返回 `(-1, -1)` 。

```python
import re

pattern = r'hello'
s = 'hello hello'
result = re.match(pattern, s)
print(result)
print(type(result))

print(result.group())
print(result.start())
print(result.end())
print(result.span())
```

* `pattern = r'hello'`中的`r`表示原始字符串，不要处理`\x`的转义字符。

> [!important]
>
> 处理正则时，正则表达式采用`r'xxx'`原始字符串。

### 正则对象

`re.compile()`可以将正则表达式的样式编译为一个正则对象[`re.Pattern()`](https://docs.python.org/zh-cn/3.13/library/re.html#re.Pattern)。这个正则对象包括的方法，与前面的相同：

* `Pattern.search()`
* `Pattern.match()`
* `Pattern.findall()`
* `Pattern.finditer()`
* `Pattern.sub()`

```python
import re

# 匹配一个或多个数字
result = re.compile(r'\d+')
print(result)
print(type(result))

src = 'a 123 b 456 c 789 d'

print(result.match(src))
print(result.search(src))
print(result.findall(src))
print(result.finditer(src))
print(result.split(src))
print(result.sub('-', src))
```
