# 参数和返回值

## 函数的参数

### 形参与实参

<img src="https://raw.githubusercontent.com/hughxusu/lesson-py/develop/images/base/Xnip2025-11-14_20-19-44.jpg" style="zoom:50%;" />

* 形参：定义函数时用于接收数据的参数。
* 实参：调用函数时传入了真实的数据。

### 位置参数

一个函数可以由多个参数，函数调用时按顺序关联形参。

```python
def subtract_list(first, second):
    if len(first) != len(second):
        return None

    result = []

    for i in range(len(first)):
        result.append(first[i] - second[i])
    return result

a = [1, 2, 3]
b = [4, 5, 6]
result = subtract_list(b, a)
print(result)
```

### 关键字参数

通过键值对形式指定实参，调用时将名称和值会关联起来，可以忽略参数顺序。

```python
result = subtract_list(first=b, second=a)
# result = subtract_list(second=a, first=b)
```

> [!note]
>
> 关键字参数只是函数调用时，传递参数的一种方式。

位置参数和关键字参数可以一起使用，位置参数一定在关键字参数之前，且实参个数必须与形参一致。

```python
result = subtract_list(b, second=a)
```

### 不定长参数

不定长参数（可变参数），传入的参数可以个数和值可以任意变化。

#### 元组形式

```python
def avg(*args):
    print(args)
    if len(args) == 0:
        return None

    sum = 0
    for i in args:
        sum += i
    avg = sum / len(args)
    return avg

result = avg(1, 2, 3, 4, 5)
print(result)
```

1. 在形参前加星号 `*args` ，获取参数时去掉星号。
2. 获取到的参数类型是元组。

#### 字典形式

```python
def create_employee(name, **kwargs):
    print(kwargs)
    employee = {
        'name': name,
    }
    
    for key, value in kwargs.items():
        employee[key] = value

    return employee

employee = create_employee('Bob', age=25, job='dev', email='bob@example.com')
print(employee)
```

1. 在形参前加两个星号 `**kwargs` ，获取参数时去掉星号。
2. 获取到的参数类型是字典。

### 参数默认值

在函数定义时，可以为参数指定默认值，调用是可以有默认值的参数形参可以不传。

```python
def create_student(name, std_id, email=None, phone=None, is_active=True):
    student = {
        'name': name,
        'std_id': std_id,
        'email': email, 
        'phone': phone,
        'is_active': is_active,
    }
    return student
student = create_student('张三', '1001', 'zhangsan@example.com')
print(student)
```

强制使用关键字参数，`*`以后的参数只能用关键字形式。

```python
def create_student(name, std_id, *, email=None, phone=None, is_active=True):
    student = {
        'name': name,
        'std_id': std_id,
        'email': email, 
        'phone': phone,
        'is_active': is_active,
    }
    return student

student = create_student('张三', '1001', phone='13800000000', is_active=False)
print(student)
```

* 强制关键字参数经常与默认参数一起使用。
* 提高代码可读性且避免参数顺序错误。

### 函数定义的规则

1. 定义函数时函数的参数顺序从左到右为：位置参数、元组参数和字典参数。
2. 如果有默认参数，要写在元组参数和关键字参数之间。如果修改默认参数，则使用关键字参数形式。

```python
def calculate_total(pack_fee, *food_fee, delivery_fee=10, **coupon):
    total = pack_fee + delivery_fee
    for fee in food_fee:
        total += fee

    for key, value in coupon.items():
        if key != 'discount':
            total -= value

    if 'discount' in coupon:
        total *= coupon['discount']

    return total


result = calculate_total(5, 10, 20, first_order=5, discount=0.8)
print(result)
```

### Python中的数据存储

Python中变量是一个指针，用于指向数据的内存地址。标准的Python解释器是用C语言编写的，这意味着每一个Python对象都是C语言结构体，该结构体不仅包含其值，还有其他信息。

<img src="https://raw.githubusercontent.com/hughxusu/lesson-py/develop/images/base/Xnip2024-04-30_17-38-33.jpg" style="zoom: 33%;" />

Python 3.4中的一个整型实际上包括 4 个部分：

* `ob_refcnt`是一个引用计数，它帮助Python默默地处理内存的分配和回收。
* `ob_type`将变量的类型编码。
* `ob_size`指定接下来的数据成员的大小。
* `ob_digit`包含Python变量表示的实际整型值。

使用`id()`函数可以查看变量的内存地址。

1. 不可变数据类型，相同值可能共享数据，也可能开辟新的存储。

```python
a = 1024
b = 2048
c = 1024
print(f'a=1024：{id(a)}')
print(f'b=2048：{id(b)}')
print(f'c=1024：{id(c)}')

a = 3.14159
b = 3.141
c = 3.14159
print(f'a=3.14159：{id(a)}')
print(f'b=3.141：{id(b)}')
print(f'c=3.14159：{id(c)}')

a = '锄禾日当午'
b = '汗滴禾下土'
c = '锄禾日当午'
print(f'a=锄禾日当午：{id(a)}')
print(f'b=汗滴禾下土：{id(b)}')
print(f'c=锄禾日当午：{id(c)}')

a = 1, 2
b = 1, 1
c = 1, 2
print(f'a=1, 2：{id(a)}')
print(f'b=1, 1：{id(b)}')
print(f'c=1, 2：{id(c)}')
```

2. 可变数据类型，即使初始值相同，也会指向不同的内存。

```python
a = [1024, 2048]
b = [1024, 1024]
c = [1024, 2048]

print(f'a=[1024, 2048]：{id(a)}')
print(f'b=[1024, 1024]：{id(b)}')
print(f'c=[1024, 2048]：{id(c)}')

print(id(a[0]) == id(b[0])) # 数组里相同的不可变类型指向相同的地址
```

3. 不可变数据类型，修改变量值，会开辟新的指针。

```python
a = 1024
b = a
print(f'a=1024：{id(a)}, b=a：{id(b)}')

a = 2048
print(f'a=2048：{id(a)}, b=a：{id(b)}')
```

4. 可变数据类型，修改变量值时，不会开辟新的指针。

```python
aa = [10, 20]
bb = aa
print(f'aa={aa}->{id(aa)}, bb={bb}->{id(bb)}')

aa.append(30)
print(f'aa={aa}->{id(aa)}, bb={bb}->{id(bb)}')
```

### 在函数中修改列表

```python
def self_add(a):
    print(f'a={a}, a_id={id(a)}')
    a += a
    print(f'a={a}, a_id={id(a)}')

b = 100
self_add(b)
print(f'b={b}, b_id={id(b)}')

c = [11, 22]
self_add(c)
print(f'c={c}, a_id={id(c)}')
```

> [!note]
>
> 对于可变数据类型，在函数中直接进行参数传递，会共享同一份内存数据。函数中对列表修改，会影响函数外的数据。

使用切片形式调用，可以创建类别的副本。不改变原列表。

```python
c = [11, 22]
self_add(c[:])
print(f'c={c}, a_id={id(c)}')
```

#### 列表参数初始化

Python函数的参数默认值只会在函数定义阶段被创建一次，之后使用是同一对象。

```python
def append_value(value, items=[]):
    items.append(value)
    return items

first = append_value('foo')
print(first)
second = append_value('bar')
print(second)
```

两次调用返回同一函数。

```python
first = append_value('foo')
print(f'first={id(first)}')
print(f'__defaults__={id(append_value.__defaults__[0])}') # 读取参数的默认值
```

如果希望使用可变默认参数，通常使用 `None`

```python
def append_value(value, items=None):
    if items is None:
        items = []
    items.append(value)
    return items
```

## 函数的返回值

除了简单的返回值，Python也可以同时返回多个值。

```python
def circle(r):
    pi = 3.14
    length = 2 * pi * r
    area = pi * r ** 2
    return length, area

length, area = circle(5)

print(f'length={length}')
print(f'area={area}')
```

1. `return length, area`返回的数据类型是元组，可以用拆包的方式接收返回值。
2. `return`也可以返回列表或字典，用于返回多个值。

> [!warning]
>
> 返回多个值时，尽量只返回一种类型数据。

### 卫语句

使用卫语句来优化函数中的条件判断。卫语句是在条件分支中直接返回结果。

```python
def get_greet_msg(time):
    if time > 5 and time < 8:
        return '亲，早上好！'
    elif time < 12:         
        return '亲，上午好！' 
    elif time < 13:
        return '亲，中午休息一会吧！'
    elif time < 17:
        return '亲，下午好！'
    elif time < 18:
        return '亲，下班了！'
    elif time < 23:
        return '亲，晚上好！'
    else:
        return '亲，深夜了，早点休息吧！'

time = int(input('请输入当前时间：'))
print(get_greet_msg(time))
```

> [!alert]
>
> 函数返回后，后续的程序不会再执行。

## 函数编写规范

1. 适当的给函数添加注释。
2. 给函数参数指定默认值时，`=`了两边不要有空格。调用时也遵循该约定。
3. PEP 8函数名称和参数的总字符数不超过79个。
4. 函数名使用蛇形命名法。
5. 不要编写太长的函数，对于Python语言来说，一个函数长度不应该超过65行。

```python
def calculate_total(pack_fee, *food_fee, delivery_fee=10, **coupon):
    total = pack_fee + delivery_fee
    for fee in food_fee:
        total += fee

    for key, value in coupon.items():
        if key != 'discount':
            total -= value

    if 'discount' in coupon:
        total *= coupon['discount']

    return total

result = calculate_total(5, 10, 20, first_order=5, discount=0.8)
print(result)
```
