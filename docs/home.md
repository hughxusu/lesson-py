# Python语言与AI编程实践

**本站网址**：https://hughxusu.github.io/lesson-py/

## 课程目标

学习Python编程语言及其相关应用，重点培养代码阅读能力与AI辅助开发能力。

## 课程内容

- 了解Python编程语言的由来与应用
- 配置Python开发环境（Trae + Miniconda）
- 掌握Python的基本语法结构，能够阅读和理解代码
- 了解计算机程序的基本运行原理
- 学习面向对象编程思想
- 学习Python常用库，如文件操作、NumPy等
- 培养与AI协作编程的能力：需求描述、代码审阅、错误定位与修正
