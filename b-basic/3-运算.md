# 变量的运算

## 数学运算



### 列表和元组拼接

列表、元组有与字符串类似的拼接操作。

```python
colors1 = ['red', 'green', 'yellow']
colors2 = ['white', 'black', 'blue']
colors = colors1 + colors2
print(colors)

colors = colors1 * 2
print(colors)

colors1 = ('red', 'green', 'yellow')
colors2 = ('white', 'black', 'blue')

colors = colors1 + colors2
print(colors)

colors = colors1 * 2
print(colors)
```

> [!attention]
>
> 字典不支持 `+` 和 `*` 操作。

## 变量输入

函数 `input()` 让程序暂停运行，等待用户输入一些文本。获取用户输入后，Python将其存储在一个变量中。

```python
message = input("请输入")
```

用户输入的任何内容 python 都解读为字符串，如果想从输入信息中获得数字，需要强制类型转换。

### 类型转换

| 函数       | 说明                                            |
| ---------- | ----------------------------------------------- |
| `int(x)`   | 将 x 转换为一个整数，x 可以为字符串和浮点数。   |
| `float(x)` | 将 x 转换到一个浮点数，x 可以为字符串和整数数。 |
| `bool(x)`  | 将 x 转换到一个布尔类型                         |

```python
price_str = input("请输入商品单价：")
number_str = input("请输商品数量：")

price = float(price_str)
number = float(number_str)

total = price * number
print(total)
```

函数嵌套调用

```python
price = float(input("请输入商品单价："))
```

布尔值的转换

```python
# True
print(bool(10))
print(bool('python'))
print(['color'])
print({ 'name': '龙傲天' })
print(bool(' '))

# False
print(bool(0))
print(bool({}))
print(bool([]))
print(bool(set()))
print(bool(None))
print(bool(''))
```

| 函数       | 说明                    |
| ---------- | ----------------------- |
| `set(x)`   | 将 x 转换为一个集合     |
| `list(x)`  | 将 x 转换到一个列表。   |
| `tuple(x)` | 将 x 转换到一个元组。   |
| `str(x)`   | 将 x 转换到一个字符串。 |

列表、元组、集合与字符串之间可以相互转换。

```python
colors = ['red', 'green', 'yellow']
print(list(str(list(set(tuple(colors))))))
```

## 赋值运算符

赋值运算符操作同样适用于字符串、列表和元组

```python
# 列表
colors1 = ['red', 'green', 'yellow']
colors2 = ['white', 'black', 'blue']

colors1 += colors2
print(colors1)

colors1 *= 2
print(colors1)

# 元组
colors1 = ('red', 'green', 'yellow')
colors2 = ('white', 'black', 'blue')

colors1 += colors2
print(colors1)

colors1 *= 2
print(colors1)
```

## 逻辑运算符

Python中逻辑运算包括：与、或和非

| 名称 | 运算符 | 逻辑表达式 | 描述                                                         |
| ---- | ------ | ---------- | ------------------------------------------------------------ |
| 与   | `and`  | `x and y`  | 如果`x`为`False`，`x and y`返回`x`的值，否则返回`y`的值。    |
| 或   | `or`   | `x or y`   | 如果`x`是`True`，返回`x`的值，否则它返回`y`的值。            |
| 非   | `not`  | `not x`    | 如果`x`为`True`，返回`False`。如果 x 为`False`，它返回`True`。 |

| 名称 | 运算符 | 逻辑表达式 | 描述                                                         |
| ---- | ------ | ---------- | ------------------------------------------------------------ |
| 与   | `and`  | `x and y`  | 只有 x 和 y 的值都为 True，才会返回 True<br />否则只要 x 或者 y 有一个值为 False，就返回 False |
| 或   | `or`   | `x or y`   | 只要 x 或者 y 有一个值为 True，就返回 True<br />只有 x 和 y 的值都为 False，才会返回 False |
| 非   | `not`  | `not x`    | 如果 x 为 True，返回 False<br />如果 x 为 False，返回 True   |

```python
# 逻辑与
day_number = 3
print(day_number >= 1 and day_number <= 5)

# 逻辑或
day_number = 6
print(day_number == 6 or day_number == 0)

# 逻辑非
is_male = True
print(not is_male)
```

> [!warning]
>
> 在 Python 中比较运算符可以连写。
>
> `day_number >= 1 and day_number <= 5 ` ==> `1 <= day_number <= 5`

and 和 or 的短路，通过左边能得到整个式子的结果，因此没必要再判断右边。

* `and` 左边为 `False`，则短路。

```python
print(20 and True)
print(20 and False)
print(20 and '')
print(20 and [])
print(20 and 100)
print(None and 100)
print({} and 100)
```

* `or` 左边为 `True`，则短路。

```python
print(False or 100)
print(20 or [])
print(None or {})
```

> [!warning]
>
> and 和 or 的运算返回值并不一定是布尔值。

## 运算符优先级

以下表格的算数优先级由高到最低顺序排列。

| 运算符                     | 描述                   |
| -------------------------- | ---------------------- |
| `**`                       | 幂 (最高优先级)        |
| `* / % //`                 | 乘、除、取余数、取整除 |
| `+ -`                      | 加法、减法             |
| `<= < > >=`                | 比较运算符             |
| `== !=`                    | 等于运算符             |
| `= %= /= //= -= += *= **=` | 赋值运算符             |
| `not or and`               | 逻辑运算符             |

运算符的结合性：先执行左边的叫左结合性，先执行右边的叫右结合性。

* Python 中大部分运算符都具有左结合性，也就是从左到右执行。
* 乘方运算符、单目运算符、赋值运算符和三目运算符例外，它们具有右结合性，也就是从右向左执行。
