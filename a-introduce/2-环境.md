## 开发工具安装

### Anaconda

下载安装 Anaconda，Anaconda 中 Python 的版本为 3.x 版。

本课程使用的Python的版本为3.x版，注意 Python 2.x 的版本与 3.x 的版本无法兼容，**不要使用 Python 2.x 版本**。

### PyCharm

是由 [ JetBrains](https://www.jetbrains.com.cn/) 公司开发的一款，非常优秀 Python 集成开发环境。

#### 集成开发环境

集成开发环境（IDE，Integrated Development Environment）——集成了开发软件需要的所有工具，一般包括：

* 图形用户界面
* 代码编辑器（支持代码补全/自动缩进）
* 编译器/解释器
* 调试器（断点/单步执行）
* 其它

<img src="https://s1.ax1x.com/2023/06/01/p9zKgPS.png" style="zoom: 40%;" />

PyCharm 包括：

* Professional：Python 开发工具，同时支持 HTML/JS/SQL开发，付费。
* Community：纯 Python 开发工具，免费。

#### JetBrains 教育许可

使用高校是学生或教师邮箱注册就可以申请许可证，注意：要使用 .edu 结尾的邮箱。
